 Employee Attrition Analysis Dashboard
 Project Overview

This project analyzes employee attrition patterns using SQL, Python, and Power BI.
The goal is to identify key factors influencing employee turnover and provide insights that can help organizations improve employee retention strategies.

The dataset contains employee information such as job role, department, income, overtime status, satisfaction levels, and attrition status.

 Objectives

The main objectives of this project are:

Analyze overall employee attrition rate

Identify departments and roles with the highest attrition

Examine the impact of overtime on employee turnover

Study employee demographics such as gender and marital status

Build an interactive Power BI dashboard for HR decision-making

Dataset

Dataset used: IBM HR Analytics Employee Attrition Dataset

Age

Attrition

Department

JobRole
